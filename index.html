Eniere's Improved default HUD for Team Fortress 2
April 03, 2014

-------------------------------------------------------------------------------------------------------------

This is not a HUD in its usual meaning, but the some fixes for standart Team fortress 2 HUD, adding HP numbers on target IDs, small ubercharge value label near the crosshair, etc. - see screenshots for more info. I do it for myself, but maybe someone will find it useful too.

-------------------------------------------------------------------------------------------------------------

How to turn on/off crosshairs/hitmarker

To disable the hitmarker, open scripts\hudlayout.res, then CTRL+F for "3.1" and change "visible" and "visible_minmode" parameter values to "0".

How to change damage position and visibility

To enable alternative layouts, just rename huddamageaccount_yellow.res or huddamageaccount_m0re.res to huddamageaccount.res (placed in resource\ui\ folder).

How to change scoreboard layout

6v6 version is enabled by default when cl_hud_minmode 1. To disable, just delete scoreboard.res file placed in resource\ui\ folder.

How to change Medic HUD layout

To enable, just rename hudmediccharge_alt.res to hudmediccharge.res (placed in resource\ui\ folder) AND rename hudanimations_tf_alt.txt to hudanimations_tf.txt (placed in scripts\ folder). Not recommended to use with Vaccinator.
